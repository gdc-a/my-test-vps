Cloudinary Setup

Setup Instructions:

  1. Go to Cloudinary Settings > Enable Unsigned Uploads.
  2. Create an Upload Preset and set it to Unsigned.
  3. In Settings > Security > Restricted Media Types, uncheck Resource List to allow listing images.
